A Course Project for Web Technnology

A web based platform that uses images or photographs to automate the process of medicinal plant identification. Users are able to input images of medicinal plant, and the algorithm will correctly identify the type of the medicinal plant.The sytem returns the desired results to the user which includes the name of the respective medicinal plant and the confidence associated with it, that signifies the matching percentage with the actual plant.
